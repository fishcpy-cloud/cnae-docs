# CNAE 文档项目

这是 CNAE 官方文档项目，使用 VuePress 构建。

## 项目简介

本项目用于管理和发布 CNAE 相关的 API 文档、使用指南和教程。

## 技术栈

- VuePress v2
- Markdown
- TypeScript

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 本地开发

```bash
pnpm dev
```

### 构建项目

```bash
pnpm build
```

## 项目结构

- `docs/`: 文档内容
  - `.vuepress/`: VuePress 配置
  - `api/`: API 文档
  - `guide/`: 使用指南
- `package.json`: 项目配置和依赖
